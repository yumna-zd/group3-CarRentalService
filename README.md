# Car Rental Services Website

## Project Overview
A car rental services website built with HTML and CSS, designed to provide users with an intuitive platform to browse and rent vehicles online.

## Features
- **Home Page**: Welcome section with featured services and call to action
- **Vehicle Fleet**: Browse available cars with detailed specifications
- **Booking System**: Easy to use rental booking interface
- **Contact Information**: Get in touch with customer support

## Technologies Used
- HTML5
- CSS3

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/yumna-zd/group3-CarRentalService.git
   ```
2. Open `index.html` in your web browser to view the website locally
